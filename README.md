# EarthKeepers

EarthKeepers - это сайт благотворительности, чтобы спасти нашу планету.

## Просмотр

Посмотреть собранный проект можно в файлах index.html, styles.css, script.js в основной директории проекта

### Установка

1. Склонируйте репозиторий.
2. Перейдите в _папку проекта_.
3. Запустите npm install.
